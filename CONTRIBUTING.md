# Как помочь проекту

Спасибо, что улучшаешь Shadowrocket-маршрутизацию.

## Локальная проверка

Перед PR запусти:

```bash
python3 build_shadowrocket.py
python3 -m py_compile build_shadowrocket.py
```

Если меняешь генератор и из-за этого изменились конфиги, коммить сгенерированные файлы вместе с изменением генератора.

## Где что лежит

- `build_shadowrocket.py` отвечает за конвертацию, порядок правил и сборку.
- `roscomvpn-shadowrocket.conf` - основной публичный профиль.
- `rules/*.list` импортируются основным профилем через `RULE-SET`.
- `roscomvpn-shadowrocket-expanded.conf` нужен для отладки парсинга Shadowrocket.

## Как заводить баг

Лучше сразу приложить:

- версию Shadowrocket и платформу;
- какой URL или файл импортировал;
- короткий кусок PacketTunnel-лога рядом с ошибкой;
- приложение или домен, который не работает.
