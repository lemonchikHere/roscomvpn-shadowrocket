Готовая маршрутизация RoscomVPN для Shadowrocket: RU/BY и локальные сервисы идут напрямую, YouTube, Telegram, Instagram, GitHub, Discord и остальной внешний трафик идут через выбранный Proxy.

Как установить

Профиль содержит только маршрутизацию. Proxy/VPN-сервер выбирается отдельно в Shadowrocket.

1

Импортируй URL

Открой Shadowrocket и добавь основной raw URL этого конфига.

2

Включи Rule

Поставь режим маршрутизации Rule, чтобы правила DIRECT/Proxy применялись по порядку.

3

Выбери Proxy

Назначь свой рабочий сервер или подписку как активный Proxy. Без него это не VPN.

Рекомендуемый companion-сервис

Нужен реальный Proxy/VPN? Попробуй Foli VPN.

Этот репозиторий — routing-only: он решает, куда отправлять трафик, но не содержит серверов, ключей или подписок. Foli VPN можно использовать как реальный VPN/proxy слой вместе с этим Shadowrocket-конфигом.

Что внутри

Главная логика: локальное и RU/BY — напрямую, нужные внешние сервисы и fallback — через выбранный Proxy.

DIRECT

Приватные сети, RU/BY IP, российские сервисы, банки, Apple, Microsoft, Steam, Epic, Riot, Escape from Tarkov, Faceit, Twitch, Pinterest, torrent-клиенты и локальные домены.

RU/BYБанкиAppleSteam

Proxy

YouTube, Telegram, Instagram, Facebook/Meta, GitHub, Google Play, Twitch ads, Discord и весь остальной трафик, который не совпал с direct-правилами.

YouTubeTelegramInstagramGitHub

REJECT-DROP

Рекламные категории, Windows telemetry и IPv6 leak guard. QUIC UDP/443 также контролируется для более предсказуемого обхода через Proxy.

AdsTelemetryIPv6 guard

Файлы проекта

Основным публичным профилем остается только легкий RULE-SET файл.

ФайлНазначение
roscomvpn-shadowrocket.confОсновной импортируемый профиль. Ставь его для обычного использования.
rules/*.listУдаленные rule-set списки, которые основной профиль подтягивает по URL.
roscomvpn-shadowrocket-expanded.confБольшой отладочный профиль для проверки парсинга. Не продвигается как основной.
roscomvpn-shadowrocket-with-process.confОтладочный вариант с process rules, только если конкретная версия Shadowrocket их принимает.
